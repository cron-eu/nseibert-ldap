Compatibility
^^^^^^^^^^^^^

- Version 3.x of eu\_ldap is not compatible with LDAP server records
  created with 2.x, instead you have to redefine your server records in
  the configuration file.

- Version 3.1 supports TYPO3 6.0 and higher.
